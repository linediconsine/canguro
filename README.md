# Canguro

## A proof of concept for a FastAPI micro service

_A common myth about the kangaroo's English name is that it was a Guugu Yimithirr phrase for "I don't know" or
"I don't understand_

_Source: wikipedia_


### Installation

```bash
gh repo clone linediconsine/canguro

cd canguro

make install
```

### Usage

```bash
make run
```
